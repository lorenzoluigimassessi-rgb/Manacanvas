Browse by art style
Active filter chips
Implementation
Each pill maps to a Scryfall query. No image analysis. Zero risk of cards disappearing. Works exactly like existing filters.
Interaction
Single select — click to apply, click again to deselect. Dropdown closes on selection. Chip appears in filter bar with ✕ to clear.
Combinable
Works with all other filters — e.g. "Dark & Gritty" + Artist "Aleksi Briclot" shows only dark art by that artist.
Extensible
New styles can be added by defining a label + Scryfall query. No code changes needed beyond the styles array.
